# Warm Ledger (GitHub Pages Prototype)

This is a static magazine prototype ready for GitHub Pages.

## Included sample pages
- `index.html` — homepage
- `page-text-photo.html` — text + basic image
- `page-video.html` — text + YouTube embed
- `page-graphics.html` — graphical storytelling
- `page-issue.html` — issue-style listing page

## Deploy on GitHub Pages
1. Push this repo to GitHub.
2. In **Settings → Pages**, set source to **Deploy from branch**.
3. Select branch: `main` (or your branch), folder: `/docs`.
4. Save; GitHub will publish the site URL.

You can replace the dummy content and warm theme variables in `assets/styles.css`.
