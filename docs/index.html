      How to read this evidence

      1) Exogenous shock

      Destination-currency changes shift remittance purchasing power based on pre-2001 destination exposure.

      2) Place transformation

      Census outcomes track changes in amenities, education, labor allocation, and household structure.

      3) Mechanisms

      RVS measures household channels: spending, coping, labor decisions, and migration behavior.

      4) Local equilibrium

      NEC shows firm-side composition, formality, employment, and productivity patterns.
